Mounting mill foundation sensors

Secure the steel base plate provided by welding or bolting it to the mill foundation.

Run armored flylead from foundation sensor to junction box. Secure flylead path to junction box using Unistrut or welded brackets.

Terminate sensor flylead in junction box. Route armored cable to motor room.
